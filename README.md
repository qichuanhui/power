# 电力应用部署

本项目提供了基于图计算模型重构与FPGA硬件加速的电力系统核心算法实现。针对传统串行计算方法在处理大规模电网时效率不足的问题，本项目将潮流计算 (Power Flow, PF) 和 状态估计 (State Estimation, SE) 转化为以点为中心的图算法，并利用 FPGA 的高并发能力，实现了从秒级到毫秒级的性能跨越。
## 潮流计算 (Power Flow - PF)
功能：计算电网各节点的电压幅值、相位角、有功功率和无功功率。
算法重构：将非线性牛顿 - 拉夫逊迭代法中的右端项计算映射为图遍历操作 (ProcessEdge, AccumulateVertex)，利用 FPGA 并行处理数万条边的复数运算。
适用场景：网损计算、静态安全分析、暂态稳定计算基础。
## 状态估计 (State Estimation - SE)
功能：在量测数据存在误差的情况下，通过加权最小二乘法 (WLS) 估计系统真实状态向量。
算法重构：针对复杂的残差计算和雅可比矩阵更新，设计专用的图计算核函数，支持变压器变比、量测权重等复杂属性的并行处理。
适用场景：实时调度、应急分析、最优潮流的基础数据清洗。
